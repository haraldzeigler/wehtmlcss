Først er det inline, deretter internal og deretter extremal css definisjoner



Eksempel på inline css er dette:

<h1 style ="color:blue;">En overskrift som er blå</h1>

En overskrift som er blå


<p style ="color:red;">Et avsnitt med rød skrift</p>

Et avsnitt som er rød skrift

Hvis du vil gjøre forskjellige ting med dette, kan du vite at alle disse parametrene er mulige å skru på i koden. Sjekk det ut og still inn for Eksempel
  • bakgrunnsfarge med background-color
  • tekststørrelse med font-size
  • font med font
