Eksempel på inline css er dette:

Tekstfarger
som for eksempel grønn, rød, blå, eller andre med koden (og her brukte jeg såkalt span klasse for å gjøre akkurat bare det ene elementet mellom <span>tæggen til den fargen)
  • tekstfarge grønn <span style = "color:green">grønn</span>
  • tekstfarge rød <span style = "color:red">rød</span>
  • tekstfarge blå <span style = "color:blue">blå</span>


Bakgrunnsfarge
background-color
  • bakgrunnsfarge med background-color
  • tekststørrelse med font-size
  • font med font

Tekstfarge og bakgrunn! Skille attributter med semikolon
som for eksempel grønn tekst med marineblå bakgrunn, rød, blå, eller andre med koden (og her brukte jeg såkalt span klasse for å gjøre akkurat bare det ene elementet mellom <span>tæggen til den fargen)
  • tekstfarge grønn med marineblå bakgrunn<span style = "color:green; background-color: aquamarine;">grønn</span>
  • tekstfarge rød <span style = "color:red">rød</span>
  • tekstfarge blå <span style = "color:blue">blå</span>
